# STM32-BMS-48Pro

基于 **STM32F103C8T6 + BQ76940** 的电池管理系统（BMS）重构项目。

本项目以学习、重构和验证电池管理系统架构为目标，围绕真实硬件平台逐步实现电池采样、告警保护、均衡控制、执行控制、CAN 通讯、FreeRTOS 多任务调度以及异常安全处理等功能。

项目当前仍在持续开发与优化中。

---

## 1. 项目简介

`STM32-BMS-48Pro` 是一个面向嵌入式学习与求职展示的 BMS 项目。

项目以 **BQ76940 电池监控芯片** 为核心，完成多串锂电池的电压、电流、温度采集，并在此基础上实现软件告警、硬件保护、均衡控制、FET 控制、CAN 状态上报以及异常安全关断流程。

当前项目重点不只是“功能能跑”，而是尽量按照实际工程思路进行模块划分和架构重构，包括：

* 驱动层与应用层分离
* 采样、保护、均衡、控制任务拆分
* FreeRTOS 多任务调度
* I2C 总线互斥访问
* 全局状态互斥保护
* CAN 状态帧与故障帧设计
* 初始化失败后的 Fail-Safe 安全处理

### 4.10 CAN 通讯

当前已实现 CAN 状态上报与故障上报。

CAN 状态帧：

| CAN ID | 内容                  |
| ------ | ------------------- |
| 0x301  | Pack 总压、电流、温度、状态    |
| 0x302  | Cell 1 ~ Cell 4 电压  |
| 0x303  | Cell 5 ~ Cell 8 电压  |
| 0x304  | Cell 9 电压、告警标志、保护标志 |
| 0x305  | BQ76940 初始化失败故障帧    |

故障帧 `0x305` 用于上报：

* 故障来源
* 主错误码
* Bring-up 失败阶段
* 底层错误码
* 重试次数
* Safe-Off 结果
* 故障标志位

### 4.11 FreeRTOS 多任务架构

当前系统已迁移到 FreeRTOS 多任务架构。

已实现任务：

| 任务              | 功能                         |
| --------------- | -------------------------- |
| BMS_SampleTask  | 电压 / 电流 / 温度 / SYS_STAT 采样 |
| BMS_ProtectTask | 软件告警与硬件保护处理                |
| BMS_BalanceTask | 自动均衡控制                     |
| BMS_ControlTask | 执行层控制                      |
| BMS_CANTask     | CAN 周期上报                   |
| BMS_AuxTask     | LED 与运行状态打印                |
| BMS_GaugeTask   | BQ34Z100 电量计预留任务           |

核心任务链路：

```text
SampleTask
    ↓
ProtectTask
    ↓
BalanceTask
    ↓
ControlTask
```

CAN 与 Aux 任务采用周期运行方式。

---

### 4.12 互斥锁与任务同步

当前已实现：

* `ctx mutex`：保护全局 BMS 状态结构体
* `i2c mutex`：保护 Soft I2C 总线
* 二值信号量：用于任务链路接力

已完成采样与均衡任务的锁粒度优化：

```text
I2C mutex：
    只保护硬件 I2C 读写

ctx mutex：
    只保护全局状态读取 / 提交

无锁阶段：
    数据换算、算法判断、状态决策
```

这样可以避免长时间持有全局锁，提高任务实时性和模块解耦程度。

---

### 4.13 初始化异常处理

已实现 BQ76940 上电 bring-up / self-test 异常处理机制。

自检内容包括：

* BQ76940 唤醒
* 基础寄存器读取
* ADC 校准参数读取
* 硬件保护参数加载
* OCD / SCD 配置
* 采样链路自检

异常处理流程：

```text
BQ76940 初始化失败
    ↓
最多重试 3 次
    ↓
最终失败
    ↓
强制 Safe-Off
    ↓
关闭 BQ76200 CHG / DSG / CP / PCHG
    ↓
尝试关闭 BQ76940 FET 与均衡
    ↓
CAN 发送 0x305 故障帧
    ↓
LED 故障提示
    ↓
进入 STOP 低功耗故障保持态
```

该流程保证系统在 BQ76940 初始化失败时不会继续创建 FreeRTOS 任务，而是优先进入安全关断状态。

## 6. 运行时串口输出

当前默认串口输出采用一行摘要形式：

```text
[BMS] P=35346mV MAX=VC1:3967mV MIN=VC6:3842mV D=125mV I=2mA T=291dC ALM=00 PROT=20 BAL=1:VC1 SYS=00
```

字段说明：

| 字段   | 含义               |
| ---- | ---------------- |
| P    | Pack 总压          |
| MAX  | 最高单体             |
| MIN  | 最低单体             |
| D    | 最大压差             |
| I    | Pack 电流          |
| T    | TS1 温度，单位 0.1°C  |
| ALM  | 软件告警标志           |
| PROT | 保护状态标志           |
| BAL  | 均衡状态与目标电芯        |
| SYS  | BQ76940 SYS_STAT |
